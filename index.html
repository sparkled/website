The Web-Based LED Sequencer

Sparkled is an easy-to-use LED sequencer that aims to reduce the barrier to entry for lighting projects.

Sparkled is designed for smaller hobbyist projects built on top of commodity hardware. For example, a 1000 pixel project can be driven at 60 frames per second running on a Raspberry Pi 3 without breaking a sweat.

Sequence, Schedule, Serve, Sparkle!
